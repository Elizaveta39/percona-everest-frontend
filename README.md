# Install bit

```bash
make install-bit
```

# Install dependencies sdfasdf

```bash
make init
```

# Running Bit  compositions

```bash
bit start
```

#  a  app (.g. )

```bash
cd apps/<app-name>
make run
```

# Run unit tests for app
```bash
cd apps/<app-name>
make test
```

# Run E2E tests for app
```bash
cd apps/<app-name>
make test-ui
```

# Build app
```bash
cd apps/<app-name>
make build
```
